# 🛡️ Ultimate 50-in-1 Graphical Cyber Security Suite & Forensic Toolkit

A comprehensive, enterprise-grade desktop security control panel engineered using **Python** and **PyQt5**. This toolkit provides an advanced compact layout with 4 columns and large fonts for optimized forensic visibility.

## 🚀 Core Features & Integrated Modules
* **OWASP Top 10 Scanners**: Advanced fuzzing for RCE, SQLi, CSRF, DOM-XSS, XXE, SSRF, CORS, and HPP vulnerabilities.
* **API & Backend Protection**: Dedicated tools for GraphQL Introspection auditing, SOAP/XML validation, and REST API security mapping.
* **Network & Wireless Forensics**: DNS Spoofing detection and Wireless Deauthentication signal monitoring.
* **Infrastructure Defense**: Reactive Anti-DDoS traffic analysis engine with real-time automatic rate-limiting and IP banning.
