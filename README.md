## ⚙️ Installation & Production Deployment

To safely execute or daemonize this graphical security framework on Debian/Ubuntu linux environments, run:

```bash
# Clone the verified repository profile
git clone https://github.com
cd cyber-security-suite

# Install core packaging modules
pip install PyQt5 pillow flask --break-system-packages

# Fire up the standalone desktop control panel
python3 desktop_app.py
```
